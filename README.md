# SigLearn — Biosignal Analyzer & Learning Platform

An interactive educational tool for understanding biomedical signals.

## Supported Signals
- **ECG** — Electrocardiogram (MIT-BIH Arrhythmia Database)
- **EMG** — Electromyography (Ninapro DB1)
- **PPG** — Photoplethysmography (BIDMC Dataset)
- **Respiration** — (BIDMC Dataset)
- **EEG** — Electroencephalography (MNE EEGBCI)

## Features (Phase 1)
1. Automatic Signal Classification
2. Interactive Signal Plot (zoom, pan, hover)
3. Peak Detection Visualization
4. Derived Parameters (HR, RR, band power, RMS...)
5. AI Explanation Engine
